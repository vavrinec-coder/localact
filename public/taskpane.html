LocalAct

Local Excel Add-in

Checking service

1. Update P&L

2. Update P&L by Departments

Uses current PL_Mapping for Summary Account.

3. Update BS

4. Update Operations

5. Formulas to Values

Convert LocalAct formulas in the active worksheet to static values.

Validation & Notes

  • Text columns are trimmed before validation and storage.
  • Each update replaces existing data for that section.
  • Missing mappings block the update.
  • Dates must be true month-end Excel dates.
